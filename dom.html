JS/ DOM Manipulation/ Selecting Elements

There are 6 methods to select elements in the DOM.
1. The getElementById method selects an element with the matching id attribute value. TO specify the id: getElementbyId("exampleId").
2. getELementsByClassName() selects elements with matching class attribute values. To specify the class name: getELementsByClassName("classname").
3.querySelector() selects the first element with the matching css selector. To specify by class name: querySelector(".classname"). To specify by id: querySelector("#exampleId").To specify by element tag name: querySelector("h1").
4. To select all elements with matching css selectors in the document use querySelectorAll().
5. getElementsByTagName() selects all elements with a matching tag name.
6.getElementsByName() selects all elements with a matching name attribute value.


Selecting an element and assigning it to a variable:
