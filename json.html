JavaScript/ JSON

JSON stands for JavaScript Object Notation. JavaScript Object Notation is a format which eases data transfer.

Parse and Stringify

When it came to JSON the first code snippets I would use centered around JSON.parse() and JSON.stringify() When you use JSON.parse() you're turning a JSON string into a JavaScript Object. When you use JSON.stringify() you're turning a JavaScript Object into a string which follows JSON notation. A JavaScript Object looks like:

const obj = {name: "Steve", age: 30, city: "Anchorage"};

A JSON string looks like:

'{"name":"Steve", "age":30, "city":"Anchorage"}'
